# Social Ops Review Loop

Use this skill after publishing to avoid mixing up two different failures:
- execution failure
- content failure

## Step 1: verify execution

Ask first:
- Did the platform actually publish?
- Did the creator-side page reflect the new post?
- Did the expected images/body land correctly?

## Step 2: verify distribution signal

Then inspect:
- initial views
- likes / comments / collections
- whether comments started at all
- whether the hook likely lost the first scroll

## Step 3: decide the next move

Choose exactly one:
1. fix execution bug
2. keep content and wait for more signal
3. rewrite hook / first screen
4. change topic angle
5. pause repeated posting density
